#캡쳐 캐비넷

###기능
1. 스크린샷 모아서 정리하기
   1.1 휴대폰으로 캡쳐한 "스크린샷"만 한 곳에 모아준다.
   1.2 모은 스크린샷을 내가 설정한 폴더에 옮겨 담는다.
   1.3 드래그 & 드롭으로 원하는 "스크린샷"을 폴더에 "담는다.


###화면
1. 모음 화면 : 내가 찍은 모든 "스크린샷"들을 볼 수 있다.
2. 폴더 화면 : 내가 만든 폴더들을 확인할 수 있다.
3. 폴더 리스트 화면 : 폴더 화면에서 "탭"으로 진입하면, 해당 폴더에 지정된 스크린샷을 3*3그리드로 볼 수 있다. 
4. 상세 이미지 화면 : 폴더 안에
